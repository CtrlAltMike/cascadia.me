Proof of concept

Hazard atmosphere

Four very small effects, one per hazard, layered on top of the existing guide hero. Nothing in the live site is touched — these pages reference ../css/base.css and ../css/components.css as-is and add a single new stylesheet, effects.css.

Open each one and let the page sit for ~30 seconds before judging. The earthquake tremor in particular only happens once every 25s.

Flood

Slow rising water-line beneath the type. ±2px, 11s, water-color at 3–7%.

Fire

Warm radial glow pulse + a horizontally drifting smoke-light hatch.

Storm

Sparse drifting flakes with a faint windblown diagonal hatch.

Quake

1px tremor on hero text every 25s; barely-there fault line at the bottom.

If approved, integration is one CSS file + four small markup adds

Each guide hero would gain one <div class="fx-layer">…</div> block right inside .guide-hero, and effects.css would be linked alongside components.css. Earthquake also adds fx-quake-target to the hero's .container. No existing rules change.
